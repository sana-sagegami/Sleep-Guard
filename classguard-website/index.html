AI搭載の居眠り検知システム

授業中の居眠りを自動検知

face-api.jsを使用した顔認識技術で、リアルタイムに居眠りを検知します。
目の開閉や頭の角度を分析し、集中力を維持するアラートを送信。

🎯

リアルタイム検知

顔検出、目の開閉、頭の角度をリアルタイムで分析。居眠りを即座に検知します。

🔔

アラート通知

居眠りが検知されると、音声アラートと通知で注意を促します。

📊

視覚的フィードバック

ステータスインジケーターで現在の状態を常に確認できます。

🔒

プライバシー保護

⚡

軽量・高速

TinyFaceDetectorを使用し、低負荷で高速に動作します。

🌐

全サイト対応

オンライン授業、Web会議など、あらゆるWebサイトで使用可能。

📱 スマホで自動撮影

居眠りが検知されると、スマホで自動的に撮影します。
PCのChrome拡張機能と連携して動作します。

  1. スマホで以下のURLを開く
    ブラウザ（Safari/Chrome）でアクセスしてください。
  2. 「📷 QRコードスキャンを開始」をタップ
    カメラ権限を許可してください。
  3. 拡張機能に表示されたQRコードを読み取る
    PCのChrome拡張機能で「検知開始」後にQRコードが表示されます。
  4. 自動接続完了
    居眠りが検知されると、スマホで自動撮影されます。

インストール方法

  1. 拡張機能ファイルをダウンロード
    下のボタンから SleepGuard.zip をダウンロードします。
  2. ZIPファイルを解凍
    ダウンロードしたファイルを解凍し、フォルダを任意の場所に保存します。
  3. Chrome拡張機能ページを開く
    chrome://extensions/ をアドレスバーに入力します。
  4. デベロッパーモードを有効化
    右上の「デベロッパーモード」をONにします。
  5. 拡張機能を読み込む
    「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。
  6. 完了！
    拡張機能アイコンから「検知開始」をクリックして使い始めましょう。
